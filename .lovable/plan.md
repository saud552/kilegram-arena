

# خطة إعادة تصميم واجهة كيليجرام (Kilegram UI Redesign)

## الهدف
إعادة تصميم الواجهات الرئيسية للعبة بأسلوب بسيط وعملي ونظيف، مُحسّن للعمل كتطبيق مصغر داخل تيليجرام (شاشة موبايل).

---

## المرحلة 1: شاشة البداية (Splash Screen)
- تحسين شاشة البداية بشعار KILEGRAM مع أنيميشن بسيط وأنيق
- إضافة شريط تحميل (Loading Bar) يعكس حالة تحميل بيانات اللاعب من Supabase
- تصميم داكن نظيف بألوان اللعبة (أزرق + أحمر)

## المرحلة 2: الشاشة الرئيسية (Home Screen)
- **الشريط العلوي:** صورة اللاعب، الاسم، المستوى، والعملات (Coins + Gems) بتصميم مُبسّط
- **شريط XP:** شريط تقدم الخبرة مع عرض المستوى الحالي
- **زر BATTLE:** زر مركزي كبير وواضح للدخول للمعركة
- **أزرار سريعة:** Create Team + Join via Code بتصميم بطاقات صغيرة
- **بطاقة الإحصائيات:** عرض Kills, Wins, Matches بشكل مُنظم
- **شريط الأخبار:** عرض آخر الأحداث والفعاليات من جدول `game_events`

## المرحلة 3: صفحة المتجر (Store)
- تبويبات واضحة: الأسلحة | الشخصيات | المعدات
- بطاقات المنتجات مع صورة/أيقونة، الاسم، السعر، وزر الشراء
- مؤشر واضح للعناصر المملوكة والمُجهّزة
- عارض النماذج ثلاثية الأبعاد للأسلحة والشخصيات (الموجودة كملفات GLB)

## المرحلة 4: صفحة الفريق (Squad/Lobby)
- عرض كود الفريق بشكل واضح مع زر نسخ
- قائمة أعضاء الفريق مع حالة كل لاعب (جاهز/غير جاهز)
- زر بدء المعركة (للقائد فقط)
- تصميم بطاقة لكل لاعب تعرض الاسم والمستوى والسكن المُختار

## المرحلة 5: صفحة الملف الشخصي (Profile)
- عرض معلومات اللاعب: الصورة، الاسم، المستوى، البلد
- إحصائيات تفصيلية: القتلات، الانتصارات، المباريات، معدل K/D
- قسم المخزون: عرض العناصر المملوكة مع إمكانية التجهيز
- سجل المباريات الأخيرة من جدول `match_history`

## المرحلة 6: التنقل السفلي (Bottom Navigation)
- تحسين شريط التنقل بأيقونات واضحة وتأثير التبديل
- 4 تبويبات: الرئيسية | المتجر | الفريق | الملف الشخصي
- إضاءة خفيفة للتبويب النشط

---

## ملاحظات تقنية
- التصميم مُحسّن لشاشات الموبايل (Telegram Mini App)
- استخدام Framer Motion لأنيميشنات بسيطة وسلسة
- الألوان الأساسية: خلفية داكنة (#0a0f1a)، أزرق (#00a6ff)، أحمر (#ff3b3b)
- البيانات تُجلب مباشرة من Supabase (profiles, inventory, match_history, game_events)
